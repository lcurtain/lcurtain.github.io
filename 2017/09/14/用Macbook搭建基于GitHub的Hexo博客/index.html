用Macbook搭建基于GitHub的Hexo博客

发表于 2017-09-14

某天突发奇想，想要找一个可以自己一个人码字的地方。现在提供这样服务的网站很多，甚至简书和知乎已经支持markdown在线写作。但是我不是一个能接受别人评论我写作内容的人，说白了就是怕喷。所以，只能自己建个窝，来安放我TM胡乱瞎写的脑洞和文字。

正好知乎搜到了如何简单快速小白建博客的教程，感觉还在我操作接受的范围，所以我就按照网上大神的步骤，建了这个博客。

所需网站：

  1. GitHub
  2. 域名购买：GoDaddy
  3. DNS解析：dnspod
  4. Hexo主题设置：NexT
  5. 图床：七牛
  6. 阅读统计：LeanCloud

##github page

github-page是一个免费的静态网站托管平台，由github提供，它具有以下特点：

  1. 免空间费，免流量费
  2. 具有项目主页和个人主页两种选择
  3. 支持页面生成，可以使用jekyll来布局页面，使用markdown来书写正文
  4. 可以自定义域名
