Hexo Seo简单优化

发表于 2017-09-18 | 分类于 Hexo

我建好了博客上传了内容，才发现其他人其实根本就不能在网上搜到我，自嗨的很开心。但是建立博客的目的不就是想让更多的人发现和评论我写的东西么，所以让搜索引擎知道和收录我的网站就很重要了。

所以我在网上找了各种大神留下的教程，自己慢慢的学着优化博客。

验证网站

在百度和谷歌上分别搜索

注意博客域名不是xxx.github.io。

如果此前没有进行过操作，应该是搜不到的，并且搜索出来的结果含有搜索引擎提交入口。

提交域名

需要在各个搜索引擎的站长平台添加自己的网站。

百度站长平台

搜狗站长平台

谷歌Search Console

原理都相同，进入搜索引擎提交入口，添加域名，选择验证网站，有3种验证方式，本文推荐采用HTML标签验证。

复制获取到的标签，打开主题next\layout\_partials\head.swig，把代码粘贴到里边。

然后终端提交更新，然后网页上的验证，就可以通过验证了。

生成站点地图

想要被搜索引擎收录，就需要站点地图，站点地图是一个网站所有链接的容器。很多网站的连接层次比较深，爬虫很难抓取到，站点地图可以方便爬虫抓取网站页面，通过抓取网站页面，清晰了解网站的架构，网站地图一般存放在根目录下并命名sitemap，为爬虫指路，增加网站重要内容页面的收录。

我们需要使用npm自动生成网站的sitemap，然后将生成的sitemap提交到百度和其他搜索引擎

安装sitemap插件

然后修改博客的配置文件_config.yml，把文件中url改为你的站点地址。

执行完之后就会在网站根目录/public生成sitemap.xml文件和baidusitemap.xml文件，其中sitemap.xml文件是搜索引擎通用的文件，baidusitemap.xml是百度专用的sitemap文件。

自动推送

next主题配置文件，将baidu_push改为true

然后你可以找到themes\next\layout\_third-party\seo\baidu_push.swig,这样每次访问博客中的页面就会自动向百度提交sitemap
